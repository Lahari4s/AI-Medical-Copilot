# 🩺 AI Medical Copilot

AI-powered healthcare assistant with OCR, medical report analysis, symptom checker, personalized diet tracking, medicine reminders, and live health tracking.

---

# 🚀 Features

- 📄 OCR medical report extraction
- 🧠 AI medical analysis
- 🍱 Personalized food tracking
- 💬 AI healthcare chatbot
- 💊 Prescription & medicine reminder
- 📊 Live health tracking dashboard
- 🧬 Disease prediction engine
- 🧠 Symptom checker
- 📁 Multi-user chat history
- 📄 PDF export

---

# 🛠 Tech Stack

- Python
- Streamlit
- OpenRouter API
- SQLite
- Plotly
- OCR (Tesseract + PyMuPDF)

---

# ▶ Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

# 🔐 Environment Variables

Create `.env`

```env
OPENROUTER_API_KEY=your_api_key
```
